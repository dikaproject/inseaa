### Installation
* composer
  ```sh
  composer install
  ```
* Database Migrate
  ```sh
  php artisan migrate - untuk migrate database
  ```
* npm - untuk menginstall tailwind dll
  ```sh
  npm install 
  ```
* npm Run dev in other terminal / terminal berbeda dengan php artisan serve
  ```sh
  npm run dev
  ```
* start web, buka link ini untuk lihat webnya
  ```sh
  php artisan serve
  ```

### DON'T FORGET TO CHANGE THE .ENV | JANGAN LUPA UNTUK MENGGANTI .ENV NYA
